**Windows usage:**

>Run setup.exe to install keyboard layout.

**BSD usage:**

>`xmodmap hieamtsrn-standard-bsd.xmodmap`

FreeBSD/GhostBSD use older version of XKeyCaps 2.46, it's not compatible with 2.47.

**Linux usage:**

>`xmodmap hieamtsrn-standard-linux.xmodmap`
